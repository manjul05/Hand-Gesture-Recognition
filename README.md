# Hand Gesture Recognition System

## Overview
A real-time Hand Gesture Recognition System built using Python, OpenCV, MediaPipe, and CNN. The system detects hand gestures through a webcam and predicts sign language gestures.

## Features
- Real-time hand detection
- Gesture recognition using CNN
- MediaPipe hand landmark tracking
- Webcam integration
- High accuracy prediction

## Tech Stack
- Python
- OpenCV
- MediaPipe
- TensorFlow/Keras
- NumPy

## How to Run

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run

python cnn_predict_with_mediapipe.py

- Validation Accuracy: 99%+
